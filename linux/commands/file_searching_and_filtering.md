### File Searching and Filtering

##### Given below are default commands in Linux which are used for file searching and filtering.
1. File Search
    ```
    #Find file.txt
    find /path -name "file.txt"
    ```
2. Fast file search
    ```
    #Fast file search
    locate filename
    ```
3. Find command location
    ```
    #Locate command location
    which python
    ```
4. Search text inside files
    ```
    #Find lines with "error"
    grep "error" logfile.log
    ```
