### User and Permissions Management

##### Given below are default commands in Linux which are used for user and permissions managament.
- User Management
    ```
    #Create user
    useradd milto
    
    #Delete user
    userdel username
    ```
- Permission Management
    ```
    #Add execute permissions to user.
    chmod u+x file

    #Remove write permissions
    chmod g-w file
    ```
