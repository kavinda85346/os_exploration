### File and Directory Management

##### Given below are default commands in Linux which are used for File and Directory management.
1. Navigating Directories
    ```
    #Showing the current directory.
    pwd

    #List file directories in detail.
    ls -l
    ```
2. Creating Files and Directories
    ```
    #Create a file.
    touch file.txt
    ```
3. Viewing File Contents
    ```
    #Display the file contents.
    cat file.txt

    #View the 10 lines of a text.
    head file.txt
    ```
4. Copying, Moving and Renaming
    ```
    #Copy contents in file_1 to file_2 directory.
    cp file1 file2

    #Move or rename file.
    mv file1 file2
    ```
5. Deleting Files and Directories
    ```
    #Delete file_1.txt
    rm file.txt

    #Delete boot 
    rmdir boot
    ```
