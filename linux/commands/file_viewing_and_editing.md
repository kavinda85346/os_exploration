### File Viewing and Editing

##### Given below are default commands in Linux which are used for file viewing and editing.
1. Display entire file content
    ```
    #View the content file_2
    cat file_2.txt
    ```
2. View first 20 lines
    ```
    #View the first 20 lines of file_2.txt
    head -n 20 file_2.txt
    ```
3. View last 20 lines
    ```
    #View the last 20 lines of file_2.txt
    tail -f 20 file_2.txt
    ```
4. Search text inside files
    ```
    #Find lines with "error"
    grep "error" logfile.log
    ```
5. Editing the file using nano
    ```
    #Editing the nano script.
    nano script.sh
    ```
6. Editing the file vim
    ```
    #Editing the vim script.
    vim script.sh
    ```
