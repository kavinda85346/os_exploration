### Networking

##### Given below are default commands in Linux which are used to view and manage network configurations.
1. Test end to end connectivity.
```
#Sending a packet from one destination to the other destination.
ping 127.0.0.1
```
2. Check the route that a packet took to reach from one destination to the other destination.
```
#Checking the destination that a packet to took.
traceroute 8.8.8.8
```
3. Display the network interfaces
```
#View all the detials realted to the network interfaces.
ifconfig
```
4. View network connections and listening ports.
```
netstat
```
