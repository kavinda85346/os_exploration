### Search and File Operations

##### Given below are default commands in macOS which are used for search and file operations.
1. File Search Commands
    ```
    #Searches for files in a specific directory recursively.
    find /Users/kavinda/Documents -name "*.txt"
    
    #Search for a string inside a file.
    grep "Hello" myfile.txt
    ```
2. File and Directory Operations
    ```
    #List files in the current directory.
    ls

    #Create a file.
    touch hello1.sh
    ```
