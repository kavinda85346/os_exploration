### System Information

##### Given below are default commands in macOS which are used to get system information.
1. Basic System Information
    ```
    #Shows macOS version details.
    sw_vers

    #Displays kernel related details.
    uname -a
    ```
2. Hardware Information
    ```
    #List all disks and partitions.
    diskutil list

    #Get a detailed information about RAM
    system_profiler SPMemoryDataType
    ```
