### System Control and Administration

##### Given below are default commands in macOS which are used for system control and administration.
1. System Information
    ```
    #View macOS version.
    sw_vers

    #View system uptime and load averages.
    uptime
    ```
2. Process Management
    ```
    #Show all running processes.
    ps aux

    #Kill a process by PID.
    kill PID 321
    ```
