### Disk and File System

##### Given below are default commands in macOS which are used for disk and file systems.
1. Disk Management Commands
    ```
    #List all disks and partitions.
    diskutil list

    #Get a detailed infor about a disk.
    diskutil info /dev/disk1
    ```
2. File System & File Operations
    ```
    #List files and directories.
    ls -la

    #Creating a current directory.
    mkdir hello
    ```
