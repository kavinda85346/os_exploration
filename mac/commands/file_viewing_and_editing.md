### File Viewing and Editing

##### Given below are default commands in macOS which are used for file viewing and editing.
1. Viewing Files
    ```
    #Viewing contents of fruits.sh
    cat fruits.sh

    #View the top ten lines of filename.txt file.
    head -n 10 filename.txt
    ```
2. Editing Files
    ```
    #Editing the script.sh file using the nano editor.
    nano script.sh

    #Editing the script.sh file using the vim editor.
    vim script.sh
    ```
