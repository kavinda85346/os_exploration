### System Utilities and Troubleshooting Commands

##### Commands given below are used for system ulities and troubleshootig commands.

- System Information and Diagnostics
    1. System details via CLI
        ```
        #Displays detailed system configuration (OS, RAM, CPU, uptime, patches).
        systeminfor
        ```
    2. Windows version information
        ```
        #Dispaly Windows version information.
        winver
        ```
- Disk and File System Utilities
    1. View disk error troubleshoot.
        ```
        #View disk error and repair them.
        chkdsk
        ```
    1. View disk cleanup.
        ```
        cleanmgr
        ```
- Network Troubleshoot
    1. Check end to end connectivity.
        ```
        ping 8.8.8.8
        ```
    2. Trace route to a destination.
        ```
        tracert 8.8.8.8
        ```
- File and Process Management
    1. View running tasklist.
        ```
        tasklist
        ```
    2. Launch File Explorer.
        ```
        explorer
        ```
