### Advanced Tools

##### Given below are default commands in Windows which are used to launch tools.

##### We can classify these commands into 10 categories and they are listed below.
1. System and Process Management
2. Disk and File System
3. File and Directory Operations
4. Network and Troubleshooting
5. User and Security Managaement
6. System Configuration and Services

##### Commands that are belong to each of these categories are listed below along with their usage.
1. System and Process Management
    ```
    #View running processes.
    tasklist
    
    #View full system details.
    systeminfo
2. Disk and File System
    ```
    #Check disk errors.
    chkdsk

    #Disk defragmentation.
    defrag C
    ```
3. File and Directory Operations
    ```
    #Manage file permissions.
    icacls "C:\Data" /grant UserName:F

    #View attributes of a file.
    attrib filename.txt
    ```
4. Network and Troubleshooting
    ```
    #Test end to end connectivity.
    ping 8.8.8.8

    #View routing information
    tracert 200.100.50.2
    ```
5. User and Security Management
    ```
    #View the current user infor
    whoami

    #View localgroup
    net localgroup
    ```
6. System Configurations and Services
    ```
    #View scheduled tasks.
    schtasks

    #View system configuration.
    msconfig
    ```
