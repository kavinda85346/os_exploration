### File and Directory Manageemnt Commands

##### Commands given below are used for file and directory managent.
1. Create a file directory.
```
#Create a file directory.
mkdir photos
```
2. Remove an empty directory.
```
#Delete an empty directory.
rmdir photos
```
3. Remove a non-empty directory.
```
#Delete a non-empty directory.
rm -r photos
```
4. Change file chanagement.
```
#Change a file directory.
cd photos
```
5. Rename a file to a new file.
```
#Change a file directory.
ren folder_1 folder_2
```
6. Create a file.
```
#Create a file.
echo. > file.txt
```
